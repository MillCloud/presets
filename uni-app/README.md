# vue

uni-app 模板，适用于小程序和移动端应用开发。

请使用 [Vue Language Features (Volar)](https://github.com/vuejs/language-tools) 和 [接管模式 Takeover Mode](https://cn.vuejs.org/guide/typescript/overview.html#volar-takeover-mode)。
